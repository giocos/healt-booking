# PrenotazioneVisitaMedica

// Descrizione
Il nostro progetto si basa su un sito CUP online per effettuare una visita medica. La pagina principale del sito prevede un’area di login per soli amministratori o impiegati, collegamenti alle rispettive sezioni, ed un API per seguire un video tutorial su “come prenotarsi”. L’utente interessato accede alla sezione “Prenotazione” compilando un form inserendo i propri dati personali, e una volta confermato verrà generato un codice QR associato alla prenotazione ed un download contenente il riepilogo della prenotazione. La prenotazione potrà essere convalidata solo ed esclusivamente da un impiegato attraverso la scansione di un codice QR o attraverso l’inserimento da tastiera di un codice esadecimale. Inoltre è possibile anche visionare lo stato della propria prenotazione inserendo il codice ad esso associato o controllare tutte le prenotazioni effettuate in un determinato giorno. Il sito web dispone di un’area dedicata all’assistenza degli utenti i quali potranno inviare una segnalazione, spiegando dettagliatamente il problema riscontrato, che verrà visionata dagli amministratori di sistema o dagli impiegati. Inoltre se l’utente ha specificato la propria email, potrà essere contattato per l’eventuale risoluzione del problema.
